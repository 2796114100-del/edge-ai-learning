# 第 2 周 · 面试题 + 投递

## Day 6-9 · 高频面试题（你当模拟面试官，一题题问，答案素材来自他的项目）

> **给 AI 老师**：下面是真实调研的高频题库，按类过。重点打磨**量化、TensorRT、CUDA 手写**这三块——最能区分"真做过"和"背概念"。引导学生从自己项目里找答案。

**量化（阶段二素材）**
- INT8 量化原理？对称 vs 非对称？权重/激活各常用哪种？
- **PTQ vs QAT** 区别、适用场景？QAT 里 Q/DQ 节点作用？
- 校准在做什么？KL/熵校准原理？校准集怎么选、要多少张？（你的答案：≥1000、entropy、batch=1）
- 为什么量化掉点？怎么定位敏感层？怎么恢复精度（per-channel、混合精度、敏感层留 FP16）？
- per-tensor vs per-channel？

**TensorRT（阶段一/二/三素材）**
- TensorRT 做了哪些优化？（层融合/kernel 自动调优/精度校准/显存复用/CUDA graph）
- 隐式 vs 显式量化（implicit vs explicit Q/DQ）？
- 动态 shape 怎么处理？optimization profile 是什么？workspace 作用？
- **怎么写自定义插件？什么情况必须写？**（你的答案：ONNX 有不支持算子时；用 IPluginV3；讲 Core/Build/Runtime）
- engine 为什么不能跨 GPU 架构/跨 TRT 版本复用？
- FP16 和 INT8 分别什么场景选？

**CUDA（阶段三素材）**
- grid/block/warp/thread；一个 warp 多少线程？（32）
- global/shared/register/constant memory 区别与延迟？
- 什么是 **bank conflict**？怎么用 padding 避免？
- **memory coalescing**（合并访存）是什么、为什么重要？
- **warp divergence**（线程束分化）怎么产生、怎么减少？
- **手写题**：reduce 归约、softmax、矩阵转置——会考现场写 kernel。（你写过 NMS 核，再补练 reduce/softmax）
- occupancy？`__syncthreads()`？`atomicAdd`？CUDA stream 并发？

**ONNX / 系统（阶段一/二素材）**
- PyTorch→ONNX 常见坑（动态轴、不支持算子、opset）？怎么用 graphsurgeon 改图？
- 前后处理放 GPU 还是 CPU？NMS 怎么加速？
- 怎么判断瓶颈是 compute-bound 还是 memory-bound（Roofline）？
- batch size 对吞吐/延迟影响？多模型/多流并发部署？

**（投机器人/自动驾驶岗加问）** ROS2 话题通信机制？实时性怎么保证？串口协议怎么设计防粘包？（你阶段四全做过）

✅ **【检查点 D9】** 每类挑 3-5 题，学生能各讲 2-3 分钟，答案能落到自己项目。
